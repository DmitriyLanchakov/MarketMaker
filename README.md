# MarketMaker
Implementation of Marketmaker trading strategy for Russian market.

* OrderBook Parameters calculation
* Trading politics calculation
* Plot strategy Map


![alt text](Demo/RawData/orderbook.gif "")

# Install and use

* Clone repo
* Run `orderbookMain.R`
